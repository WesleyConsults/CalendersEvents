# Calenders Events

A modern, responsive marketing website for **Calenders Events** — Accra's premier venue for dining, live music, karaoke, and private events.

## Tech Stack

- **React 19** + **TypeScript**
- **Vite 6** (bundler)
- **Tailwind CSS v4** (styling)
- **Motion** (`motion/react`) — animations
- **Lucide React** — icons

## Getting Started

```bash
npm install
npm run dev
```

The app runs at `http://localhost:3000`.

## Build

```bash
npm run build
```

## Notes

- This is a **fully static frontend** — no backend, no API keys, no environment variables required.
- All content (events, menu, gallery) is hardcoded in `src/App.tsx`.
